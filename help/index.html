Settings

Various settings you can toggle. When you type a valid command the text will change colors, you must then press "ENTER" to confirm.

  • show letters

    Show note letters on the keyboard

  • hide letters

    Hide note letters from the keyboard

Keywords

Include these keywords in addition to chords or scales in your query.

  • play

    Begin playing the notes.

  • root

    Only show the root notes.

Keyboard

Keyboard shortcuts and commands.

  • ⇧

    Transpose the query up by 1 semitone.

  • ⇩

    Transpose the query down by 1 semitone.

  • RETURN

    Clear the query and reset notes.

Chords

Chord notes appear as soon as you type them in. To show chords start by typing a root note followed by a series of modifiers.

  • C

    A simple C major chord.

  • Cm

    Make it minor by adding an 'm'.

  • Cm6

    All standard modifiers are available too, e.g. 6, M7, etc.

  • BbM7

    You can select different roots like Bb, C#, Gb, etc.

  • Fm6(b13)(#11)(+5)

    You can add accidentals by typing (b, #, +, -) followed by a number.

  • Dbdim7

    You can also use text modifiers like 'dim' and 'aug'.
